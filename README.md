# NumPy Tutorial üìä

A comprehensive, hands-on tutorial for learning NumPy - the fundamental package for scientific computing in Python.

## üìñ Description

This repository contains a series of Jupyter notebooks that cover NumPy from basics to advanced topics. Each notebook is self-contained with clear explanations, working code examples, and practice exercises.

Whether you're a beginner just starting with data science or an experienced developer looking to deepen your NumPy knowledge, this tutorial has something for you.

## üöÄ Getting Started

### Prerequisites

- Python 3.8 or higher
- pip (Python package installer)

### Installation

1. Clone this repository:
```bash
git clone https://github.com/yourusername/numpy-tutorial.git
cd numpy-tutorial
```

2. Install the required packages:
```bash
pip install -r requirements.txt
```

3. Launch Jupyter Notebook:
```bash
jupyter notebook
```

4. Navigate to the `notebooks/` folder and open any notebook to begin learning!
